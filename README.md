# GoodBOT

A not very Good discord bot, this was my first ever Discord bot...

## Prerequisites

* NodeJS 10.5 or above

### Installing

Execute the following code to install dependencies.

```
npm install
```

customize util/config.js to your particular requirements

```
node app
```
